## System/Model Parameters

**Mass**
- m = 1 *kg* = 1e-3 *t*

**Spring Stiffness**
- k = 10 *N/m* = 10-e3 *N/mm*

**Damping**
- d = 1 *Ns/m* = 1e-3*Ns/mmm*

**Gravity**
- g = 9.81 *m/s²* = 9.81e3 *mm/s²*
